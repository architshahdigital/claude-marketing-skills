# Citability Scorecard

Score each lever 0 (absent) · 1 (partial) · 2 (strong). Anything under 2 is a rewrite target. A page scoring 8–10 is genuinely citable.

| Lever | 0 | 1 | 2 |
|-------|---|---|---|
| **Answer-first structure** | Answer buried or missing | Answer somewhere in the section | Direct answer in first 1–2 sentences of each section |
| **Extractable units** | Wall of text | Some lists | One idea/sentence; definitions, steps, comparisons in lists/tables |
| **E-E-A-T evidence** | Anonymous, no sources | Named author OR some sources | Named credentialed author + primary sources + dates + first-hand data |
| **Specificity** | Hedges & generalities | Mixed | Concrete numbers, named tools, real examples throughout |
| **Freshness + scope** | Undated, partial coverage | Dated OR complete | Visible update date + full question space covered (defs, edge cases, FAQ) |

## Fast heuristics
- Read any section's first sentence alone — does it answer the query without the rest? If not, rewrite.
- Could an AI lift a 2-sentence quote that stands on its own? If not, add extractable units.
- Every number should trace to a source or the author's own data.
